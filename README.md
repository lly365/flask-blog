# Flask Blog

此项目为练习 Flask 项目开发的实战作品

## 使用

### 克隆本仓库

### 安装依赖包

```bash
pip install < requirements.txt
```

### 修改配置文件，位于根目录 config.py

### 初始化数据库

```bash
python manage.py db init
python manage.py db migrate
python manage.py db upgrade
```

### Shell

```bash
python manage.py shell
```

### Web

```bash
python manage.py runserver
```
